# 🪩 Inimigos do Fim - Software Requirements Specification (SRS)

Este repositório contém a documentação arquitetural e de engenharia de requisitos para o projeto **"Inimigos do Fim"**, um aplicativo focado no mapeamento, filtragem e monitoramento da vida noturna em tempo real.

O foco deste material é demonstrar a capacidade de traduzir regras de negócio complexas em estruturas lógicas de software, modelagem de dados e processos visuais antes da etapa de codificação.

## 🏗️ Minhas Contribuições na Arquitetura

Como principal responsável pela estruturação do sistema, desenvolvi de ponta a ponta:

* **Elicitação de Requisitos:** Definição de Requisitos Funcionais (RF) e Não Funcionais (RNF).
* **Modelagem de Banco de Dados (DER):** Estruturação do Diagrama Entidade-Relacionamento, resolvendo cardinalidades complexas (N:M) para o sistema de Check-in, Usuários, Eventos e Estabelecimentos.
* **Mapeamento de Processos (BPMN):** Desenho completo da jornada do usuário utilizando diagramas de atividades (BPMN), cobrindo desde a autenticação até a lógica de destaque de anúncios pagos integrados a um gateway externo.
